# Representation Passport

The representation passport reports three independent dimensions:

## Source Dimension

- source count;
- claim count;
- traceable claims;
- unsupported, contradicted, and partially supported claim IDs;
- quantifier warnings.

## Representativeness Dimension

- cluster count;
- covered, weakened, distorted, and omitted clusters;
- omitted salient clusters;
- minority retention count;
- procedural issue retention count;
- condition preservation count.

## Authorization Dimension

- authorization status;
- permitted and prohibited use;
- missing authorization information;
- required actions;
- required disclosures;
- triggered rule IDs.

## No Single Score

MANDATE intentionally avoids a single ethics or representativeness score. A single number would hide whether the main problem is evidence, representation, or authorization.

The passport can be exported as JSON, Markdown, or HTML. HTML is designed to be simple enough for a later PDF conversion step.
